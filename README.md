# Personal Portfolio - Omer Salah

A modern, responsive personal portfolio website built with **React**, **TypeScript**, and **Vite**. This project showcases my skills, experience, and certifications while providing a professional interface for potential collaborators to get in touch.

- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views.

- **Contact Form:** Integrated with a backend service to handle email submissions with client-side validation and real-time feedback.
- **Environment Management:** Securely managed API endpoints using `.env` files for different environments (Development vs. Production).
- **Clean UI:** Professional layout using modern design principles, featuring custom-styled sections for Skills, About, and Certifications.

## üõ†Ô∏è Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/OmerSalah-Dev/my-personal-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-personal-website
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Environment Setup

Create a `.env` file in the root directory and add your backend API URL (this file is ignored by Git for security):

```text
VITE_API_URL=https://api.omersalah.com
```

### Running the App

Start the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
